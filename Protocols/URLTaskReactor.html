URLTaskReactor

public protocol URLTaskReactor : Reactor

This protocol provides the means to react to a URLTask object.

It also inherits from the Reactor protocol.

  • shouldExecute(after:from:with:) Extension method

    Default implementation of Reactor.shouldExecute. If the result type and task type are not the expected URLTask types then shouldExecute returns false.
